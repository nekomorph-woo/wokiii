# 维度报告模板

维度报告按 Phase 2 各维度输出。编号遵循 [report-numbering.md](report-numbering.md)。

```markdown
**需求/决策**：...
**缺失层**：{哪一层没有覆盖}
**影响**：...

...

## 统计

- 需求/决策总数：{N}
- ✅ 完整追溯：{N}
- 🔴 完全缺失：{N}

```
