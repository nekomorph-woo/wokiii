HITL 使用指南

HITL（Human-In-The-Loop）是人类反馈进入管道的唯一入口。

添加备注

  1. 在文档内容中选中文本，出现「添加引用」按钮
  2. 点击按钮，引用被收集到备注面板
  3. 选择备注类型：决策 / 疑问 / 建议
  4. 输入内容，点击「添加」

备注状态

待处理新建备注，等待处理
已应用已被采纳并用于修改文档
已解决修改后确认无问题
已拒绝确认不采纳
已搁置暂时搁置，后续处理

点击备注卡片上的状态标签可以切换状态。

应用备注

  1. 点击「选择应用」进入选择模式
  2. 勾选需要处理的备注
  3. 点击「生成 Prompt」复制结构化文本到剪贴板
  4. 粘贴到 Claude Code 中执行，或直接运行 /wok-apply-remarks
  5. Skill 修改文档后备注状态变为「已应用」，展示修改摘要和影响

过滤器说明

重点关注已应用（待确认）+ 待处理（待 skill 处理），已应用在前
全部所有状态，含已拒绝和已搁置

状态流转规则

  • 待处理 → 只能由 /wok-apply-remarks 变为「已应用」；用户可手动搁置
  • 已应用 → 用户确认后变为「已解决」
  • 已解决 / 已拒绝 / 已搁置 → 可重新打开
